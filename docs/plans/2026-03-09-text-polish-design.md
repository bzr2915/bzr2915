# 文字润色设计：研究方向页 + 个人简介第二段

日期：2026-03-09

## 目标

以吴晓波《大败局》的写作风格润色两处中文内容，解决"太绕"的问题。

## 范围

1. `_pages/about_zh.md` 第二段（招人段）
2. `_pages/projects_zh.md` 全文

## 风格原则

1. **句长控制**：主力句20字以内，最长不超过40字
2. **场景先行**：每节以具体画面开头——人、地点、动作
3. **隐喻一次性**：用了就走，不反复回扣。去掉"锤子""指挥家"等贯穿全文的比喻
4. **术语消化**：方法名不逐一罗列，融入"问题→尝试→效果"的叙事。只保留MARIO、ATOM、Atelier三个名字
5. **去哲学化**：不用"肌理""消融""既定之物"等抽象词
6. **结尾一句收**："未完成的翻转"整节砍掉，改为一句话点题

## 个人简介第二段

- 第一句直接亮出意图
- 用具体行为描述代替抽象品质
- 技术态度用一句对比说清
- 结尾行动号召，干脆利落
